# leaflet-challenge

## Project Description
 - This assignment uses JavaScript and Visual Studio to look create a map using earthquake data. An HTML website is created using 1 file that was taken from the website "https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php". Visual Studio and D3 was used to import the file and display the map as required. 
## Summary of Contents
### Map
 - TileLayer created for map
 - Connects to geojson API using D3
 - Markers with size corresponding to earthquake magnitude
 - A legend showing the depth and their corresponding color
### Data Points
 - Data points scale with magnitude level
 - Data points colors change with depth level
 - Each point has a tooltip with the Magnitude, the location and depth
 - All data points load in their correct locations 
## Getting Started
 - This assignment uses Visual Studio, JavaScript, and D3 with the obtained data and creates the required map
